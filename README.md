## Dodge Food (Canvas) — готовый проект

Это простая браузерная игра на **HTML + CSS + JavaScript** (рисование через **`<canvas>`**).

### Запуск

- **Самый простой способ**: открой `index.html` двойным кликом в браузере.
- **Через локальный сервер** (иногда удобнее для разработки):
  - В PowerShell из папки проекта:

```powershell
python -m http.server 8000
```

Потом открой в браузере `http://localhost:8000`.

### Управление

- **Стрелки** `←` / `→`: движение
- **P**: пауза

### Правила

- +1 очко за каждый “фрукт”, который пролетел ниже игрока (ты уклонился)
- Столкновение с любым фруктом завершает игру
- **Рекорд сохраняется** между сессиями через `localStorage`
